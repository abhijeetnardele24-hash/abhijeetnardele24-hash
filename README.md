## Hey, I'm Abhijeet 👋

I'm a CS undergrad at **VIT Pune** who genuinely enjoys the full spectrum — from designing pixel-perfect UIs to soldering ESP32 boards at 2am. I care deeply about *how* software feels to use, not just whether it works.

Right now I'm deep in:
- 🧠 Building an **AI-powered SaaS platform** (the kind I wish existed)
- ☁️ Getting dangerous with **AWS architecture**
- 🔩 Experimenting with **WebAssembly** for performance-critical web apps

When I'm not pushing commits, I'm probably disassembling old hardware and making it do something it wasn't designed to do.

## Projects I'm Proud Of

### 🏗️ Land Registry Management System
A full-stack decentralized land records platform. Tackled the real-world problem of opaque property ownership with a clean audit trail and role-based access control.  
→ [`land-registry-management-system-deploy-final`](https://github.com/abhijeetnardele24-hash/land-registry-management-system-deploy-final)

### 🔐 Telegram OTP Door Lock
IoT meets messaging — an ESP32-based smart lock that authenticates via a Telegram bot. One of those projects that just works, and feels like magic every time.  
→ [`telegram-otp-door-lock-LDM`](https://github.com/abhijeetnardele24-hash/telegram-otp-door-lock-LDM)

### 🔤 N-Gram Text Predictor
Built a language model from scratch using N-gram statistics — no libraries, just math and code. Great for understanding what LLMs are actually doing under the hood.  
→ [`N-Gram-Text-Predictor-`](https://github.com/abhijeetnardele24-hash/N-Gram-Text-Predictor-)

### 📁 File Version Control (OOP in C++)
A mini VCS implementation using core OOP principles. Think git, but built from scratch in C++ as a deep-dive into data structures and system design.  
→ [`OOPSCP-File-Version-Control`](https://github.com/abhijeetnardele24-hash/OOPSCP-File-Version-Control)

### 📲 XTransfer & Airdrop App (Flutter)
Cross-platform file transfer apps built with Flutter. Fast, clean UI, no-nonsense UX.  
→ [`XTransfer-Flutter`](https://github.com/abhijeetnardele24-hash/XTransfer-Flutter) · [`-airdrop-app`](https://github.com/abhijeetnardele24-hash/-airdrop-app)
